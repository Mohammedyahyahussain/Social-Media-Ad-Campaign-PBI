# Social-Media-Ad-Campaign-PBI

## Overview
This project analyzes large-scale social media advertising data to evaluate campaign performance, identify inefficiencies, and optimize budget allocation using Power BI.

## Objectives
- Measure campaign effectiveness using CTR and conversion metrics
- Analyze demographic performance across age and gender
- Identify inefficiencies in ad spend
- Optimize ROI through data-driven recommendations

## Dataset
- ~213 million impressions dataset
- Includes:
  - Impressions
  - Clicks
  - Conversions
  - Spend
  - Demographics (age, gender)

## Key Features
- Built interactive Power BI dashboard for real-time insights
- Implemented KPIs:
  - Click Through Rate (CTR)
  - Conversion Rate
  - Cost per Conversion
- Used DAX for advanced calculations and segmentation
- Drill-down analysis across demographic groups

## Analysis Performed
- CTR and conversion analysis across campaigns
- Demographic segmentation (age + gender)
- Funnel analysis (Impression → Click → Conversion)
- ROI and cost efficiency evaluation

## Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

## Key Insights
- Male users (30–34) showed highest conversion efficiency
- Female users (45–49) had high spend but low returns (inefficient segment)
- Significant drop-offs observed between click and conversion stages
- Certain campaigns generated impressions but failed to convert → negative ROI

## Recommendations
- Reallocate budget from low-performing demographics to high-performing segments
- Improve ad creatives for underperforming groups
- Optimize funnel to reduce drop-offs
- Focus spend on high CTR + high conversion segments
